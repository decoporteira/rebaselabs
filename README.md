# Projeto para o Rebase Labs.

Aplicação web para importar um .cvs para um banco de dados usando Docker.

## Tecnologias:
- Docker
- PostgreSQL
- Ruby
- PostgreSQL
- Javascript
- HTML
- CSS

## Clonar o projeto: 
```
git clone git@github.com:decoporteira/rebaselabs.git
```
## Criar os containers: 
```
docker-compose up
```
## Endpoints: 
```
/tests
```
```
/tests/token_do_exame
```
Importar dados de um .csv
```
/import
``` 
## Tests: 
```
docker-compose up
rspec
```
